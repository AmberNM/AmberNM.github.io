Students identify a societal problem, identify solutions, define project solutions, develop project objectives, conduct feasibility analysis, establish organizational group structure to meet project objectives and develop formal specifications. Students make formal technical project presentations and develop web documentation.

## Workforce Development II (CS 411W)

Students write professional and non-technical documents and continue the development of the project defined in CS 410. Written work is reviewed and returned for corrective rewriting. Students will design and develop a project prototype, and demonstrate the prototype to a formal panel along with delivering the formal product specifications and a draft formal grant proposal. This is a writing intensive course.

public void completeRegistration()
{
    //All text fields must be 3 characters or longer
    String uName = userName.getText().toString().trim();
    if (userName.getText().toString().trim().length() > 2
        && email.getText().toString().trim().length() > 2
            && pass1.getText().toString().trim().length() > 2
            && pass2.getText().toString().trim().length() > 2
            && pass1.getText().toString().equals(pass2.getText().toString())
    ) {
        final PAUtils.Modal ackHipaa = new PAUtils.Modal("HIPAA Acknowledgement",
                "By clicking 'OK', you are allowing PatientAdvocate to collect your electronic healthcare records.\n\n*** Required to use the PatientAdvocate app",
                "OK");
        ackHipaa.build(moveToRegisterPage.this);
        ackHipaa.builder.setOnDismissListener(new DialogInterface.OnDismissListener() {
            @Override
            public void onDismiss(DialogInterface dialog) {
                if (ackHipaa.result) {
                    PAUtils.Modal appHipaa = new PAUtils.Modal("HIPAA Acknowledgement",
                            "Thank you for registering as a new user with PatientAdvocate!",
                            "OK");
                    appHipaa.build(moveToRegisterPage.this);
                    appHipaa.builder.setOnDismissListener(new DialogInterface.OnDismissListener() {
                        @Override
                        public void onDismiss(DialogInterface dialog) {
                            PAUtils.move2Page(moveToRegisterPage.this, MainActivity.class);
                        }
                    });
                    appHipaa.show();
                } else {
                    final PAUtils.Modal declineHipaa = new PAUtils.Modal("HIPAA Acknowledgement",
                            "With your decline of the HIPAA Acknowledgement, you will not be able to use the PatientAdvocate app.\n\nClick 'OK' to go back to the Login screen or 'Cancel' to stay on the Registration screen.",
                            "OK");
                    declineHipaa.build(moveToRegisterPage.this);
                    declineHipaa.builder.setOnDismissListener(new DialogInterface.OnDismissListener() {
                        @Override
                        public void onDismiss(DialogInterface dialog) {
                            if (declineHipaa.result)
                            { PAUtils.move2Page(moveToRegisterPage.this, MainActivity.class); }
                        }
                    });
                    declineHipaa.show();
                }
            }
        });
        ackHipaa.show();
        PAUtils.createUser(userName, pass1, MainActivity.class);
        /*PAUtils.createNewUser(userName.getText().toString(),
            email.getText().toString(),
            pass1.getText().toString(),
            pass2.getText().toString(),
            phone.getText().toString()
        );*/

    }
    else
    {
        Toast.makeText(moveToRegisterPage.this,
                "Please ensure that all required fields are filled in. All fields must be three characters or longer.",
                Toast.LENGTH_SHORT
        ).show();
    }
}
